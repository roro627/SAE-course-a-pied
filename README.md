# La Course

Bienvenue sur le projet "La Course". Ce projet est un site web dédié à la course à pied, offrant des conseils, des programmes d'entraînement, des recommandations nutritionnelles et des informations sur l'équipement adapté pour les coureurs de tous niveaux.


## Pages principales

- **index.html** : Page d'accueil en français, présentant le site et ses fonctionnalités.
- **debuter.html** : Conseils pour débuter en course à pied.
- **equipement.html** : Recommandations sur l'équipement de course à pied.
- **nutrition.html** : Conseils nutritionnels pour les coureurs.
- **objectif.html** : Informations sur les objectifs de course à pied.
- **programmes.html** : Programmes d'entraînement pour différents niveaux.
- **mentionlegales.html** : Mentions légales du site.

Toutes les pages sont disponibles en français et en anglais.
